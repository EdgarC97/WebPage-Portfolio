Arbol en consola

Descripción: Aplicacion que inicialmente que permite al usuario ingresar la altura de un arbol grafico y posteriormente se puede visualizar la representacion del mismo con asteriscos, cada linea del arbol representa el numero escrito y su tronco está conformado por 2 asteriscos solamente si el valor ingresado es igual o mayor a 5. Tecnologias y herramientas utilizadas: Para la realización de esta aplicacion se utilizó el lenguaje Python 3.11 y tambien, como editor de código fuente, la herramienta Visual studio code.
